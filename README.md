# IP-Data
基于全球ASN数据和部分厂商的公示数据提取的IDC和公有云服务IP，相邻或包含的IP段进行了合并。  
这些IP通常不会分配给终端用户使用，可用于辅助人机检查等场景。  

数据每天检查一次，如有更新会自动提交。
